~~ GUESS THE NUMBER ~~

This is a console-based Python game where the player attempts to guess a random number between 1 and 100.
The game gives feedback (higher or lower) after each guess.
The player has a limited number of guesses and is awarded a score for how quickly they guessed the number.

----------------------------------------------------------------------------------------------------------

Requirements:

Python 3.x must be installed.
No additional libraries are required.

----------------------------------------------------------------------------------------------------------


How to run the game:

1. Download or clone the project files.
2. Open terminal and navigate to the project folder.
3. Run the script using: python guess_the_number.py
4. The game will guide you through what to do.

NOTE: The secret number is within the range 1-100

----------------------------------------------------------------------------------------------------------


Features:

- Player name input
- Random number generation
- Input validation using try/except
- Attempt limit
- Score based on remaining attempts
- Option to play again
- Option to play as new player
- Leaderboard tracking total score, games played and fewest guesses per player
- Leaderboard saved to .json file
- Local leaderbord.json allows local leaderboard to be maintained across multiple runs
